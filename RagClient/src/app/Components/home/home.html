INTRODUCTION:

This a sample application that tests the implementation of AI with angular and ASP.net Web API. It has Document Search and Documents Bulk Insert, Images Search, Images Ai Insight, Images Bulk and Single Insert and Movies Bulk Insert and Search.

All three modules, Documents, Images and Movies have a start Analysis button that start the calculation of float Embeddings.

Images

There are two pages of images

  • Images Insertion, List and Semantic Search
  • Generate Images

Images Insertion, List and Semantic Search:

This module provides options to insert images in database, start image embeddings and image insight embeddings after insertion and display the list of images. This module also has options to display images in slideshow and get insight of individual image on click. This module aslo has search functionality, which first computes the similarity of images against the query and display the relevant images.

Images Generation:

With this module image can be generated based on prompt entered

Documents

There are two pages of Documents

  • Document Insertion and List
  • Chat Bot

Document Insertion and List:

This module provides options to insert documents in database, start document embeddings after insertion and display the list of documents.

Chat Bot:

This module first computes the similarity of all the documents against the query and then sends the user query alongwith previous messages and the searched documents to openrouter to get the answer of that query.

Movies

There are two pages of Movies

  • Movies List, Paging and Semantic Search
  • Config Movies

Movies List, Paging and Semantic Search:

This module provides displays all the movies pagwise. This module also provides a way to search the movies. it first computes the similarity of movies against user query and then displays the list of relevant movies. There are two types of search, in one type similarity is computed on seperate embedding of all the fields and then results are computed against weight of each field while in other type similarity if computed for embeddings of movie against a single combined string detail.

Config Movies:

This module gives options to update the weights of different fields in search, options for inserting the movies in database and option for starting the calculation of Movie Embeddings.
